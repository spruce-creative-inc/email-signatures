Spruce Email Signatures

Instructions

Gmail (in browsers)

  1. Go to settings gear icon > See all settings button
  2. Scroll down to Signature
  3. On this website, highlight all content from your signature from name through logo.
  4. Copy paste into signature edit box. All styling should copy over properly.

HTML Signature

Some email clients need the actual html to style properly. Follow these easy steps

  1. Click the Copy as Code button for your signature, it will add the html code to your clipboard
  2. Find the signature settings in your client
  3. Paste it in code signature box if it exists

If you are not sure how to set it up for your specific email client, check in with Adam and he will get you sorted.
